# 🐍🧊 Snake 2048

لعبة ثعبان مكوّن من مكعبات تحمل أرقام **مضاعفات الـ2** (2، 4، 8، 16…). الرأس يحمل
أكبر رقم، يأكل المكعبات الأصغر أو المساوية له، وتُدمج الأرقام المتشابهة مثل **2048**.
الرسم بأسلوب **2.5D إيزومتري** على Canvas، تعمل **فردي بلا إنترنت** و**جماعي عبر الشبكة**،
وقابلة للتثبيت **كتطبيق (PWA)** ومتعدّدة اللغات.

> `Snack2048-game-local-network`

## الفكرة
- الثعبان سلسلة من المكعبات، كل مكعب خلف الآخر بفجوة صغيرة، مرتّبة **تنازلياً** من الرأس للذيل.
- يأكل **بالرأس فقط** المكعبات التي رقمها **≤ رقم الرأس**.
- المكعب المأكول يُدرَج في مكانه الصحيح حسب الترتيب، ثم يحدث **دمج تسلسلي**: أي مكعبين
  متجاورين متساويين يندمجان إلى الرقم التالي (وقد يتسلسل الدمج كما في 2048).
- المكعبات تكبر تدريجياً مع الرقم، والأرقام الكبيرة تُختصر تلقائياً (`K / M / B / T …`).

## الواجهة الرئيسية (المنيو)
عند الفتح تظهر شاشة البداية وتحتوي:

| العنصر | الوظيفة |
|--------|---------|
| **اسم اللعبة** + 🌐 | بجانب الاسم زرّ اللغات (درب‑داون بحث بكل اللغات). |
| **حقل الاسم** | اكتب اسمك (يُحفظ تلقائياً). |
| **📲 ثبّت التطبيق** | يظهر على المتصفحات الداعمة لتثبيت اللعبة كتطبيق. |
| **👤 / 👥** | تبديل بين **فردي** و**جماعي** (أيقونتان). |
| **ابدأ اللعب** | يبدأ الوضع الفردي. |

في الوضع **الجماعي**: زرّ **العب أونلاين** بجانبه **قفل** 🔓/🔒 + قسم **الانضمام برمز**.

أعلى الشاشة أثناء اللعب يوجد **HUD** قابل للطيّ:
- **إحصائياتك** (درب‑داون): أعلى رقم، النقاط، الطول، الوقت، الأفضل، المرحلة + **شريط الطاقة**.
- **المتصدّرون** (درب‑داون): مطويّ يعرض مرتبتك فقط، ومفتوح يعرض **نافذة 7 لاعبين** (3 فوقك + أنت + 3 تحتك).

## طريقة اللعب الأساسية
1. وجّه الثعبان نحو المكعبات على الأرض.
2. كُل ما هو **≤ رقم رأسك**؛ ما هو أكبر تمرّ من خلاله دون أكل.
3. اجمع أرقاماً متشابهة لتندمج وتكبر (2→4→8→…).
4. تجنّب **حدود الخريطة** (الخروج منها = موت) و**مناطق الخطر**.
5. في الأونلاين: نافِس اللاعبين وكُل أجسادهم وتسلّق المتصدّرين.

## التحكّم

### الحاسوب
| المفتاح | الوظيفة |
|---------|---------|
| **الماوس** | توجيه الثعبان نحو المؤشّر. |
| **الضغط المطوّل / Space** | تسارع (يستهلك الطاقة). |
| **الأسهم / WASD** | توجيه بديل. |
| **1–5** | إرسال إيموجي. |
| **E** | استعمال شحنة الجائزة (×2). |
| **Q / F / R** | قوى BinAref (مضاعفة / سرعة / رادار). |
| **ESC** | إيقاف مؤقت. |

### الهاتف
- **عصا تحكّم لمسية** تظهر أينما تضع إصبعك للتوجيه.
- **أفقي:** زرّ **⚡** للتسارع.
- **عمودي:** لا يوجد زرّ — **دفع العصا بقوة = تسارع**، وتخفيفها = إيقاف.
- **الإيموجي**: أزرار مرقّمة (عمودية على اليمين في الوضع العمودي) قابلة للنقر أثناء الحركة.
- **زر الرجوع** أو تصغير التطبيق → إيقاف مؤقت.

## القوى ومكعبات الخريطة
المكعبات الخاصة تظهر كـ**بطاقات مسطّحة متوهّجة** على الأرض:

| البطاقة | الرمز | الأثر |
|---------|-------|------|
| السرعة | ⚡ | تضاعف سرعتك **3 ثوانٍ** (شعاع يخرج من تحت المكعب). |
| المضاعفة | ×2 | تضاعف **كل** أرقامك فوراً. |
| النصف | ÷2 | تقسم أرقامك على 2، والرقم **2 يُدمَّر** (يختفي). |
| الرادار | 📡 | يكشف **أقرب اللاعبين** حولك **8 ثوانٍ** بمؤشّرات على حافة الشاشة مع المسافة. |

الطعام العادي أرقام صغيرة (2/4/8…)، وبعضه **متحرّك** ببطء.

---

## الخريطة والبيئة
- **أنماط خرائط** متغيّرة: **مربّع، دائري، مثلّث، متاهة**.
- **الحواجز** (صناديق مرتفعة) **لا تقتل** — تمنع الدخول فقط (تدفع الرأس للخارج).
- **الخروج من حدود الخريطة = موت**.
- **مناطق الخطر** (دوائر حمراء نابضة): كل **0.5 ثانية** يسقط آخر مكعب من ذيلك ويتحوّل
  لمكعب ساكن على الأرض يأكله أي أحد. تستمر حتى يبقى **الرأس فقط**، ثم **مهلة ثانية واحدة** —
  إن لم تخرج، تخسر. (ولا تأكل وأنت داخل المنطقة.)

---

## نظام الطاقة والاندفاع
- **شريط طاقة** أسفل الإحصائيات هو العدّاد المعتمد.
- يُستهلك أثناء الاندفاع، ويُعاد ملؤه **خلال 10 ثوانٍ** عند التوقّف.
- إذا **نفد الشريط** فلا اندفاع ولا شعاع حتى يُعاد ملؤه جزئياً.
- شعاع السرعة (جسيمات + نفّاثة توهّج) يُرسم **تحت المكعبات** كأنه ينبعث من أسفلها.

---

## نظام المراحل والميداليات
المراحل **لا علاقة لها بأرقام المكعبات** مهما كبرت — تتقدّم عبر الميداليات فقط:

> من يبقى **5 دقائق متواصلة في المركز الأول** يحصل على ميدالية **وتنتقل الخريطة للتالية**.

| الزمن المتواصل في الصدارة | الميدالية |
|---------------------------|-----------|
| 5 دقائق | 🥉 برونزية |
| 10 دقائق | 🥈 فضية |
| 15 دقيقة | 🥇 ذهبية |
| 20 دقيقة | 💎 أسطورية |
| 25 دقيقة | 🔥 خرافية |
| 30 دقيقة | ♾️ لا نهائية |

إذا تغيّر المتصدّر يُصفَّر عدّاد الـ5 دقائق. (يمكن ضبط `REIGN_STEP` للاختبار.)

## الفوز والجائزة
عند بلوغ **♾️ اللانهائية** تنتهي الجولة، وتظهر **شاشة الفائز** فيها اسمه وزرّ "ابدأ اللعب".
الفائز يبدأ الجولة التالية ومعه **3 شحنات مضاعفة** (×2) تظهر جانبياً، تُستعمل بالنقر أو بحرف **E**.

## الإيقاف المؤقت
- **ESC** أو **زرّ الرجوع** في الهاتف أو **تصغير التطبيق** → شاشة إيقاف فيها **استئناف** و**الخروج للقائمة**.
- عند العودة بعد الخروج من التطبيق:
  - إن كان **لاعب آخر قد أكلك** أثناء غيابك → **تخسر مباشرة**.
  - وإلا → تستأنف وتواصل اللعب.

## الوضع الجماعي (أونلاين)
يعمل عبر **PeerJS + WebRTC** (خادم إشارة سحابي)، فيتصل لاعبان من أي مكان بكود الغرفة.

### الغرف
- **القفل مفتوح 🔓 = غرفة عامة**: بنقرة واحدة تدخل الغرفة المشتركة (أول داخل يصبح المضيف، ومن بعده ينضم تلقائياً).
- **القفل مغلق 🔒 = غرفة خاصة**: يظهر **رمز الغرفة فوراً** (في مكان خانة الانضمام نفسه) لترسله لأصدقائك، ثم تبدأ كمضيف.
- **الانضمام برمز**: الصق الرمز (زر "لصق") ثم "انضمام".

### إشعارات ودردشة
- إشعار جانبي **"👋 لاعب انضم/غادر"**.
- دردشة **رموز** تطفو فوق الثعبان.

### القتال
- **رأس أكبر يأكل الأصغر** عند التصادم رأساً برأس.
- **رأسان متساويان** → تصادم ودفع، ومن يستعمل **السرعة** يدفع الآخر للخلف.
- **عضّ الجسم**: لمس رأسك لمكعب **أصغر** من رأسك يأكله، وما **خلفه** ينفصل كمكعبات ساكنة
  يأكلها الجميع؛ المكعب **المساوي** لا يُؤكل (مملوك) فتعبر من خلاله.

### نقل الاستضافة
إذا خرج المضيف، يُختار **المتصدّر** مضيفاً جديداً **تلقائياً وبصمت** (اختيار حتمي بالنقاط ثم المُعرّف)،
ويعيد الباقون الاتصال؛ وإن لم يبقَ أحد تُغلق الغرفة.

### اللعب على شبكة محلية بلا إنترنت
لتشغيل الجماعي **بلا إنترنت** على نفس الواي‑فاي عبر خادم محلي:
1. `npm install peer`
2. `node server.js`
3. في `game.js` اضبط: `LAN = { enabled: true, host: "IP-جهاز-المضيف", port: 9000, path: "/" }`
4. افتح اللعبة على الأجهزة عبر عنوان IP الجهاز (خادم ملفات ثابت).

## قوى BinAref الخاصة
إذا كان اسم اللاعب **BinAref** (بأي حالة أحرف)، تظهر لوحة قوى على **يسار الشاشة** يستعملها
**في أي وقت وبلا حدود**:

| القوة | الزرّ |
|------|------|
| ×2 مضاعفة | Q |
| ⚡ سرعة | F |
| 📡 رادار | R |

النظام مبني كقائمة `ABILITIES` — أي قوة مستقبلية تُضاف تظهر له تلقائياً.

## اللغات
- زرّ 🌐 بجانب اسم اللعبة يفتح **درب‑داون بحث** بكل اللغات (بحث بالاسم المحلي/الإنجليزي/الدولة).
- ترجمات كاملة لـ**16 لغة** (عربي، إنجليزي، إسباني، فرنسي، ألماني، برتغالي، روسي، صيني،
  هندي، ياباني، كوري، تركي، إندونيسي، فارسي، أردو، إيطالي)، وبقية اللغات تظهر بالإنجليزية حتى تُترجَم.
- دعم اتجاه **RTL** تلقائياً، ويُحفظ اختيارك.
- الإضافة سهلة جداً: أضف كائناً في `I18N` داخل `lang.js`.

## التثبيت كتطبيق (PWA)
- ملفات `manifest.json` + `sw.js` + `icon.svg` تجعل اللعبة **قابلة للتثبيت** وتعمل **أوفلاين** بعد أول فتح.
- على **أندرويد/كروم/إيدج**: زرّ "📲 ثبّت التطبيق".
- على **آيفون (Safari)**: القائمة ← "إضافة إلى الشاشة الرئيسية".
- يتطلّب **HTTPS** (يعمل على GitHub Pages؛ لا يعمل بفتح الملف عبر `file://`).

## التصميم التقني

### البنية
| الملف | الوصف |
|------|-------|
| `index.html` | الهيكل: Canvas + HUD + القوائم + عناصر التحكّم. |
| `style.css` | كل التنسيقات + تجاوب الهاتف + الوضعان الأفقي/العمودي. |
| `game.js` | المحرّك الكامل (حلقة اللعبة، الرسم، الفيزياء، القتال، الشبكة). |
| `lang.js` | جدول الترجمات + قائمة اللغات. |
| `sw.js` / `manifest.json` / `icon.svg` | طبقة الـPWA. |
| `server.js` | خادم PeerJS محلي اختياري (لعب LAN بلا إنترنت). |

### الرسم 2.5D الإيزومتري
- إسقاط `2:1`: `screenX = (x - y)·SCALE`، `screenY = (x + y)·0.5·SCALE`.
- كل مكعب = وجه علوي (معيّن) + وجهان جانبيان بدرجات إضاءة مختلفة + حواف + لمعة + ظل أرضي.
- ترتيب الرسم بالعمق (`x + y`) بأسلوب الرسّام (Painter's Algorithm).
- الكاميرا تتبع الرأس، والتكبير **متجاوب** مع حجم الشاشة (`SCALE` بين 17 و27).

### الحركة والجسم
- مسار (Path) يخزّن آثار الرأس؛ كل مكعب يوضع على بُعد تراكمي خلف الرأس.
- التوجيه عبر تحويل اتجاه الشاشة إلى اتجاه العالم (`unproject`) ثم التفاف محدود السرعة.

### الشبكة (نموذج النجمة)
- **المضيف يملك العالم** (الطعام/القوى/الحواجز/الخطر/الخريطة/الميداليات) ويبثّ الحالة ~20 مرة/ثانية.
- **كل لاعب يحرّك ثعبانه محلياً** (استجابة فورية) ويبثّ جسده؛ القتال يُحسب محلياً ويُبلَّغ عبر المضيف.
- معرّفات العناصر تتيح المزامنة، ونقل الاستضافة حتمي عند خروج المضيف.

### الإعدادات
كل القيم القابلة للضبط (السرعة، الحجم، التباعد، فترات الخطر/السرعة/الرادار، الميداليات، عدد المكعبات)
في كائن **`CONFIG`** أعلى `game.js`.

## التشغيل والنشر

### محلياً
افتح `index.html` مباشرة (الوضع الفردي يعمل بلا إنترنت). أو خادم محلي:
```bash
python -m http.server 8000
# ثم افتح http://localhost:8000
```

### GitHub Pages (موصى به للأونلاين والـPWA)
1. ارفع كل الملفات إلى المستودع.
2. **Settings → Pages → Branch: main / root**.
3. شارك الرابط `https://username.github.io/Snack2048/`.
