# Snake 2048 🐍🧊

لعبة ثعبان مكوّنة من مكعبات تحمل أرقام مضاعفات الـ2 (2، 4، 8، 16...).
الرأس يحمل أكبر رقم، ويأكل المكعبات الأصغر أو المساوية له، ثم تُدمج الأرقام
المتشابهة مثل لعبة 2048. الرسم بأسلوب **2.5D إيزومتري** على Canvas.

> **المرحلة الحالية:** لاعب واحد محلي (بدون بوتات / بدون أونلاين بعد).

## كيف تشغّلها محلياً

ببساطة افتح `index.html` في المتصفح — لا يحتاج أي بناء أو تثبيت.

أو شغّل خادماً محلياً (اختياري):

```bash
# داخل مجلد المشروع
python -m http.server 8000
# ثم افتح: http://localhost:8000
```

## التحكّم
- **حرّك الماوس** (أو إصبعك على الجوال) لتوجيه الثعبان.
- كُل المكعبات التي رقمها **≤ رقم رأسك**.
- المكعبات المتشابهة تندمج تلقائياً إلى الرقم التالي.

## النشر على GitHub Pages

1. ارفع الملفات إلى مستودع على GitHub.
2. من إعدادات المستودع: **Settings → Pages**.
3. اختر الفرع `main` والمجلد `/ (root)` ثم احفظ.
4. سيظهر رابط مثل `https://username.github.io/Snack2048/` — شاركه والعب.

## الملفات
| الملف | الوصف |
|------|-------|
| `index.html` | الهيكل وواجهة HUD والقوائم |
| `style.css` | التنسيقات |
| `game.js` | محرّك اللعبة (الحركة، الأكل، الدمج، الرسم) |

## الإعدادات
كل القيم القابلة للضبط (السرعة، الحجم، عدد المكعبات...) موجودة في
كائن `CONFIG` أعلى ملف `game.js`.
